# React Dashboard App

Simple React dashboard with login system, sidebar, user management, transaction management.

## Features
- Login / Logout
- Protected routes
- Sidebar menu with submenus
- Dashboard, User CRUD, Transaction CRUD
- Collapsible sidebar
- TailwindCSS styling
- Axios setup ready

## Install
```bash
npm install
npm run dev
```
